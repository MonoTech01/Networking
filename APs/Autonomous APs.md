# Connecting an AP to WLC

# Step 1: Preparation
- 1 switch
- 1 AP
- 1 console cable
- 2 RJ45 (one for Switch - AP, one for AP - computer)
- a laptop
- WLC

# Step 2: Setup
- Turn off Wifi Adapter
- Set up an IP address in the same network with the default IP address of an AP
- Go to the AP's GUI
